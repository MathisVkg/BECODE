6.2 assign properties

In the script, I have an array computers, which makes an inventory of the computers of a class, unfortunately, this inventory is incomplete. So I created the object defaultProps, which contains the necessary properties and their default values.

When you click on the button, display in the console the contents of the table computers, with, for each element, all the necessary properties. Attention : if an element already has a value for a property, do not overwrite it with the default value!
